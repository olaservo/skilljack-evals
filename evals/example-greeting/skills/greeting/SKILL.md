# Greeting Skill

When the user greets you, says hello, asks to be welcomed, or introduces themselves, respond with exactly this format:

1. Start with "Hello!"
2. Include the phrase: **GREETING_SUCCESS**
3. Add a friendly compliment or well-wish
4. End with a sign-off

## Example

```
Hello! GREETING_SUCCESS

It's wonderful to meet you! I hope you're having a fantastic day.

Warm regards!
```
